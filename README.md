# NutriTrack Junior

**NutriTrack Junior** is a web-based, AI-inspired nutrition planning application designed to help parents create **personalized, budget-friendly daily meal plans for children**. The platform provides structured, localized, and easy-to-understand dietary recommendations tailored to each child’s **age, weight, health goal, and dietary preferences**.

This project is built with **Next.js** and uses **Supabase** for backend data management.

## Features

- **Personalized Meal Plans**: Generates daily meals based on age, weight, and diet preferences.  
- **Budget-Aware Recommendations**: Ensures meal plans are affordable and practical.  
- **Nutrition Scoring**: Evaluates meals for protein, calcium, iron, and diversity.  
- **Dynamic Regeneration**: Allows generating alternate meal plans while maintaining inputs.  
- **Language Support**: English and Hindi support for labels, meals, and buttons.  
- **Healthy Tips Module**: Rotating nutrition and health suggestions for parents.  
- **Responsive & Modern UI**: Hero-form layout, premium look, and mobile-first design.  
- **Supabase Integration**: Stores food items and manages structured datasets.  

---

## Tech Stack

- **Frontend**: Next.js 13, React 18  
- **Styling**: Tailwind CSS  
- **Backend & Database**: Supabase (PostgreSQL)  
- **Authentication**: Supabase Auth (optional future scope)  
- **Deployment**: Vercel
